@par Resource Editor Description 

This setup file will install the Resource Editor user interface (UI) Application on your computer.
The UI allows the user to develop his own screens using the controls provided by the library.

@par Hardware requirements

  - Processor: Minimum 1.0 GHz , Recommended 2.0 GHz.
  - RAM : 1 GB 
  
@par Supported Operating Systems

 - Windows XP, Windows Vista, Windows 7

@note
 - Make sure you have the latest service pack and critical updates for the version of Windows that you are running. 
   These can be downloaded from Microsoft update website. 
 - Microsoft .NET Framework 2.0 must be installed on the machine 
 - Microsoft Internet Explorer 6.0 with Service Pack 1 must be installed. 
 - VC++ 2005 Runtime must be installed on the system.
